- TimeStampedModel abstract base class for created_at/updated_at timestamps

- ChecklistTemplate with JSON-based items, OneToOne with Apartment
- Inspection model for scheduling, status tracking, overall rating
- InspectionItem for individual checklist results with severity levels
- Photo model for inspection documentation
- Subscription model for owner billing
- Django Admin registration with inlines, filters, search
- Default checklist fixture (German-language)
