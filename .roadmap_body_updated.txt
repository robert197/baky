## How the agent picks the next task

1. Read this issue
2. Find the first unchecked `- [ ]` item where **all issues after `←`** are closed
3. That is your next task — open it, read the requirements, implement it
4. After completing: close the issue, check it off here, move to step 2

**Rule**: Never start an issue whose dependencies are not closed. If multiple items are unblocked, pick the first one listed (top-down order matters).
